# Project Title: Telecom Churn Intelligence with Census Income Integration

- Built an end-to-end churn prediction pipeline by cleaning telecom customer data, removing leakage variables, integrating U.S. Census income data, and engineering predictive features including tenure, income quartiles, and price sensitivity.
- Developed an A/B modeling framework comparing telecom-only features with telecom plus income features and trained Logistic Regression and HistGradientBoosting models for churn prediction.
- Achieved ROC-AUC 0.8441 and PR-AUC 0.6487 with HistGradientBoosting and produced model-driven customer segmentation insights to support data-driven churn reduction and targeted retention strategies.
